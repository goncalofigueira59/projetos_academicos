# Portfólio Acadêmico - Projetos do meu percurso académico

Este repositório contém projetos desenvolvidos durante a minha formação: 
 - Curso Técnico Profissional de Eletrónica e Telecomunicações (2018 - 2021)
 - Curso Técnico Superior de Instalações Eléctricas e Automação Industrial (2021 - 2023)
 - Licenciatura em Eletrónica e Mecânica Industrial (2023 - 2027)

## Projetos Incluídos:
 > Projeto de Aptidão Profissional 
- Estação de Soldadura de 3 patamares
 
- Projeto Integrado em Automação Industrial - Elevador Vertical com seletor de cores de tampas
- Arquitetura e Protocolos de Comunicação - Rede ITED de uma pastelaria
- Projeto Integrado em Instalações Elétricas - Rede de Energia e ITED de uma planta industrial
- Projeto Temático em Programação Aplicada
- Projeto Temático em Eletrónica e Instrumentação - MicroRato
- Projeto Temático em Concepção e Produção Assistida por PC - Suporte Bicicletas
- Projeto Temático em Eletrotecnia Industrial - Sistema de seguimento solar com MPPT e BMS e estudo fotovoltaico prático
- Projeto Temático em Automação Industrial

## Objetivo:
Demonstrar as competências técnicas adquiridas ao longo do percurso académico, servindo como portfólio para oportunidades de estágio e emprego.

Cada projeto inclui documentação, esquemáticos, código e relatórios quando aplicável.
